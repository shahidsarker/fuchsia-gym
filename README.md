# Fuchsia Gym

A Pokémon Gym Page created during the NYCDA Web Development Fellowship combining code from multiple Personal Pokédex projects to showcase the information for all group members.

## Requirements

- Include a gym name
- List information on Pokémon from all members
- Highlight a "Gym Leader"
- Highlight the gym's badge
- Display information about the gym its values

## Built With

* [Bootstrap](http://www.dropwizard.io/1.0.2/docs/) - The web framework used
* [Sass](https://maven.apache.org/) - Style Sheet Language
* [Koala](http://koala-app.com) - GUI App for Sass to CSS Compilation
* [VSCode](https://code.visualstudio.com) - Source code editor
* [axios](https://github.com/axios/axios) - Promise based HTTP client
* [Pokéapi](https://pokeapi.co) - RESTful Pokémon API
